Questionnaires illimités

Créez des questionnaires complets avec sections, 10 types de questions (choix unique/multiple, texte libre, Likert, date…) et questions obligatoires.

📱

Collecte mobile sans internet

Les enquêteurs scannent un QR code et remplissent les formulaires sur leur smartphone Android. Les données sont sauvegardées localement et synchronisées dès qu'une connexion est disponible.

📍

GPS automatique NOUVEAU

Les coordonnées GPS sont capturées automatiquement à chaque réponse terrain. Visualisez tous les points de collecte sur une carte interactive OpenStreetMap.

🤖

Analyse IA NOUVEAU

Obtenez une synthèse intelligente de vos résultats : points saillants, signaux de vigilance, qualité des données — en un clic, sans compétences statistiques.

📊

Analyses intégrées

Statistiques univariées, tableaux croisés, test chi², corrélations, régression logistique, évolution temporelle — tout dans l'application, export Excel inclus.

📥

Import Excel / CSV NOUVEAU

Importez vos données existantes en 3 étapes : upload, configuration des colonnes (groupes, indicateurs, textes libres), import direct en base.

Synchronisation cloud
